# SPEC: Channelyze — анализ и структурирование Telegram-канала AIстрология

> Статус: **ФИНАЛЬНОЕ ТЗ** (запечатано 2026-05-25)

---

## 1. Цель проекта

Из экспорта Telegram-канала **AIстрология** (`result.json`, 189 сообщений) автоматически извлечь все услуги, построить структурированный прайс-лист и интерактивный тест для новых подписчиков.

**Результаты:**
- `output/services.json` — структурированный каталог услуг (правится вручную при необходимости)
- `output/price.html` + `output/price.pdf` — прайс-лист в фирменном стиле
- `output/quiz/index.html` — интерактивный тест для GitHub Pages

## 7. Детали реализации по этапам

### Этап 1 — `parse.py`

**Логика:**
1. Читает `result.json`, фильтрует `type: "message"` с непустым текстом
2. Принудительно включает посты с ID из списка: `[41, 45, 51, 52, 53, 54, 55, 56, 57, 84, 85, 165, 170, 171, 178, 179, 185, 194, 195, 197, 199]`
3. Собирает текст из строк и text_entities в единую строку
4. Отправляет каждый пост в Claude API с промптом на извлечение:
   - `name` — название услуги
   - `description` — краткое описание (2–3 предложения)
   - `price` — цена в рублях (число или null)
   - `duration` — срок выполнения (строка или null)
   - `format` — формат (письменный, голосовой, онлайн и т.д.)
   - `target_audience` — для кого
   - `category` — одна из 5 финальных категорий
5. Сохраняет в `output/services.json`

**Формат `services.json`:**
```json
[
  {
    "id": "trojnoj-portret",
    "name": "Тройной портрет",
    "category": "Самопознание",
    "description": "...",
    "price": 2500,
    "duration": "1–2 дня",
    "format": "письменный",
    "target_audience": "...",
    "source_post_ids": [194, 195]
  }
]
```

### Этап 2 — `generate_price.py`

- Читает `services.json`
- Группирует по `category`
- Генерирует `price.html`: заголовок канала, 5 секций с карточками
- Каждая карточка: название (крупно), описание, цена + срок
- WeasyPrint → `price.pdf`

### Этап 3 — `generate_quiz.py`

**Структура теста — 3 блока:**

**Блок 1 — Контекст (2–3 вопроса):**
- Что сейчас происходит в жизни
- Какая сфера требует внимания
- Что беспокоит прямо сейчас

**Блок 2 — Намерение (1–2 вопроса):**
- Ради чего пришли: понять себя / получить прогноз / разобраться в ситуации / найти ресурс

**Блок 3 — Формат (1 вопрос):**
- Как комфортнее работать: текстовый разбор / голосовая консультация / самостоятельно

**Логика scoring:**
- Каждый ответ добавляет баллы к одной или нескольким услугам
- В финале — услуга с максимальным баллом + короткое объяснение почему

**Реализация:** чистый HTML + CSS + Vanilla JS, без бэкенда, GitHub Pages.

## 8. Что НЕ входит в MVP

- Личный кабинет / авторизация
